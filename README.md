# NutritionFacts

## Sobre o projeto

Este projeto foi desenvolvido como parte dos exercícios do **FreeCodeCamp (FCC)**, para conclusão da certificação: **Design Responsivo para a Web**.

A proposta foi recriar uma tabela de informações nutricionais (Nutrition Facts Label), achei o design minimalista bonito e estou deixando salvo aqui no meu GitHub.
